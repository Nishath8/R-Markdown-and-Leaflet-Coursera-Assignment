# R Markdown and Leaflet – Coursera Assignment

## Assignment Overview
This project was created as part of a Coursera peer-graded assignment on **R Markdown and Leaflet**.

The goal of the assignment is to:
- Create a web page using R Markdown
- Display an interactive map using the Leaflet library
- Publish the web page online and submit the public URL

---

## Project Description
The web page includes:
- The date of creation
- An interactive Leaflet map
- A marker showing a location in Bangalore, India
- Zoom and pan functionality
